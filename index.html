About Me

Hello, I am Steven, a web designer/developer that specializes in building functional and responsive websites. Currently based out of Charleston, SC. I spend most of my time with my amazing wife and business partner Angela along with Watson, our spoiled Schnauzer/Yorkee mix.
You can usually find us either enjoying one of the many Charleston iconic areas or studying up on the latest trends in design at a local cafe. Currently I am proficient in the following languages:
  • HTML
  • CSS
  • JavaScript
  • jQuery
  • ColdFusion
And I have some experience with, and currently increasing my skills in:
  • Node.js
  • React
  • Redux
  • Python3
I primarily program using the following concepts:
  • Responsive Web Design (mobile first)
  • DRY (don't repeat yourself) concept
  • Responsive images/text
  • Object oriented programming
  • Agile methodoligy
I use GIT to maintain version control of all of my personal projects.

Take a minute and check out my latest projects below and contact me about your new project so we can discuss how we can improve more than just your website together!
